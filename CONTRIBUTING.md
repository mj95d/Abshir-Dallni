# المساهمة في Abshir-Dallni

## المتطلبات
- Node.js 20+
- PostgreSQL (اختياري للتطوير المحلي)

## إعداد المشروع
- تثبيت: `npm ci`
- تشغيل التطوير: `npm run dev`
- البناء: `npm run build`

## أسلوب العمل
- استخدم فروعًا وصفية: `feature/…`, `fix/…`, `chore/…`
- رسائل الالتزام على نمط Conventional Commits:
  - `feat: …`, `fix: …`, `docs: …`, `chore: …`
- افتح Pull Request مع وصف واضح، وطريقة التحقق.

## معايير الكود
- Typescript صارم، واستخدم Zod للتحقق من المدخلات.
- لا ترفع أسرارًا؛ استخدم `.env` محليًا فقط.
- التزم بمكونات UI القائمة و`utils` الموجودة.

## الاختبارات والـ CI
- تأكد من نجاح `npm run check` و`npm run build` محليًا.
- الـ CI يبني تلقائيًا على كل `push` و`PR`.

## المراجعة
- اذكر التأثيرات على الواجهة/الخادم وقاعدة البيانات.
- اربط التذاكر/القضايا ذات الصلة.
