# Abshir-Dallni

منصة دلّني (Dallni) تساعد المستخدمين على التنقل بين الخدمات الحكومية السعودية رقميًا بكفاءة، عبر واجهة ويب حديثة وواجهة خادم مبنية على Node/Express، مع قاعدة بيانات PostgreSQL وطبقة بيانات باستخدام Drizzle ORM.

## المزايا
- مساعد ذكي ثنائي اللغة (عربي/إنجليزي) يعتمد على `openai`.
- فحص تسريبات البريد باستخدام واجهة `LeakCheck` العامة.
- إدارة الاستفسارات المحفوظة، الأجهزة، والإشعارات الأمنية.
- واجهة أمامية React + Vite مع مكونات UI جاهزة.
